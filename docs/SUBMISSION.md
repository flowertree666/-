# 作品提交材料

## 作品简介

“智游 Agent”面向个性化自由行用户，将分散的景点开放信息、时令特征、近期热度和客流趋势组织为可追溯的动态景点库，再结合旅行天数、偏好和节奏，自动输出多套可落地路线。作品强调“有来源、有时间戳、可降级”的可信信息体验，并通过多目标规划兼顾景点覆盖率、出行强度和空间效率。

## 团队介绍模板

- 产品负责人：需求调研、产品设计、答辩统筹
- 智能体/算法负责人：画像抽取、规划策略、评测
- 后端负责人：Java 服务、MySQL、数据管道、高德 API
- 前端负责人：Vue 3 页面、交互与可视化
- 数据与测试负责人：数据授权、清洗、质量验证、演示测试

请替换为真实姓名、专业、分工和贡献比例。

## Demo 视频脚本（约 3 分钟）

1. 0:00—0:20：痛点与产品定位。
2. 0:20—0:50：输入杭州、3 天、自然/人文偏好和适中节奏。
3. 0:50—1:30：展示画像理解与三套方案切换。
4. 1:30—2:05：展开每日时间线、游玩时长、距离、客流等级与更新时间。
5. 2:05—2:35：介绍合规数据管道、高德路径规划和过期降级。
6. 2:35—3:00：架构、创新点与未来扩展。

## 验收指标建议

- 路线天数吻合率 100%，每日时间预算超限率 0%
- 景点来源与更新时间展示率 100%
- 路线重复景点率 0%（库存不足时明确提示）
- 规划接口 P95 小于 2 秒（不含外部 API 冷请求）
- 人工抽检开放时间准确率不低于 95%，过期数据自动标记率 100%
