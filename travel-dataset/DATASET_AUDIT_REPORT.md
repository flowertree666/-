## 2. 质量问题

- 北京 16 个 POI 尚未取得经过人工确认的开放、票务或预约规则直达页，只能将对应内容作为规划建议，不能作为实时事实。
- 来源 URL 审计覆盖 145 条来源记录、84 个唯一 URL；105 条来源记录当前可访问，40 条不可访问或超时。
- 92 条来源记录只指向官网首页，仍需补充具体规则页。
- 杭州 `HZ-S-20260716-002`、`HZ-S-20260716-005` 为占位式公众号 URL；`HZ-S-20260716-017` 当前返回 404，均已降级为不可信来源。
- `HZ-001-04` 仅由上述不可信来源支撑，已从向量索引排除。系统不会用它回答当前夜游活动、喷泉是否取消等实时问题。

HTTP 不可达也可能由反爬、网络环境或临时故障造成，因此只作为复核信号，不等同于来源内容必然错误。
