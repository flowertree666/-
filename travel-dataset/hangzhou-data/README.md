## 口径与注意事项

1. 坐标采用 GCJ-02，经度在前。
2. 开放时间、票价、预约和交通信息优先采用景区/博物馆官方页面及文旅局官网。
3. 动态公告包含 `valid_until`，超过有效期后不得作为近期推荐依据。
4. 页面未提供明确发布日期的长期规则使用 `published_at: null`。
5. `source_ids` 用于将 POI/知识片段追溯到来源；`field_evidence` 提供字段级溯源。
6. `content_type` 区分：static_intro / opening_rule / ticket_rule / transport / visitor_service / route_advice / dynamic_notice。

## RAG 使用建议

混合检索：关键词 + 向量 + `poi_id` 过滤 + `valid_until` 过滤 + 官方来源优先。
实时余票、当天开放状态和临时活动不可由本数据集直接推断，应引导用户查看官方公众号或最新公告。
